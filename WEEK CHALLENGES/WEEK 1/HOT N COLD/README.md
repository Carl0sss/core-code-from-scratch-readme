# How to convert temperatures?
```
Read the temperature
1. If the temperature is in Fahrenheit then:
2. Use the formula: C = 5/9*(F-32), where F is the previous temperature read
3. Print the result, in this case is C
4. If the temperature is in Celsius then:
5. Use he formula F = (C*9/5)+32, where C is the previous temperature read
6. Print the result, in this case would be F
7. End
```
