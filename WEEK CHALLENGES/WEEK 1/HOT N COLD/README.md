# How to convert temperatures?
```
Read the temperature
1. If the temperature is in Fahrenheit then:
2.1 Use the formula: C = 5/9*(F-32), where F is the previous temperature read
2.2 Print the result, in this case is C
3. If the temperature is in Celsius then:
3.1 Use he formula F = (C*9/5)+32, where C is the previous temperature read
3.2 Print the result, in this case would be F
4. End
```
